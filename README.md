Lab 1 — 3-Tier Architecture
Описание:
В работе реализована простая трёхзвенная архитектура:
клиент (консольное приложение)
сервер (Flask)
база данных (SQLite)

Клиент отправляет запросы на сервер по HTTP, сервер обрабатывает их и работает с базой данных.

Функционал:
-вход пользователя (логин/пароль)
-роли: admin, moderator, viewer
-создание задач (только admin и moderator)
-просмотр задач

Как запустить:
-Запускаем сервер:
 python server.py
-Инициализируем базу (в браузере):
 http://127.0.0.1:5000/init
-Запустить клиент:
 python client.py
